iOS App Installation

Install the native Windsurfer Tracker app on your iPhone via TestFlight.

What is TestFlight?

TestFlight is Apple's official app for testing apps before they're released on the App Store. It's free and safe to use.

1

Install TestFlight

If you don't already have TestFlight, download it from the App Store:

Get TestFlight from App Store

2

Join the Windsurfer Tracker Beta

Tap the button below to join the beta and install the app:

Join TestFlight Beta

What happens:
  1. TestFlight will open
  2. Tap "Accept" to join the beta
  3. Tap "Install" to download the app
3

Configure the App

Open the Windsurfer Tracker app and:

  1. Tap Select Event to choose your event
  2. Enter your name (e.g., "John" or "S42")
  3. Enter the event password (ask the race organisers)
  4. Tap Start Tracking
  5. Allow location access when prompted (select "Always")
4

Verify It's Working

You're all set! Your position will appear on the race map at wstracker.org - select your event to see the live tracking map.

The app will track your position every 10 seconds while running.

During the Race

  • Keep the app running - it will continue tracking in the background
  • Lock screen is OK - tracking continues when the screen is locked
  • Request Assistance - if you need help on the water, tap the "Request Assist" button
  • Battery - consider bringing a power bank for long races

Troubleshooting

Position not showing on map

Check:

  • Location permission is set to "Always" in Settings → Privacy → Location Services → Windsurfer Tracker
  • You have selected the correct event
  • The event password is correct (ask race organisers)
  • You have mobile data or WiFi connection

TestFlight link not working

Make sure you:

  • Have TestFlight installed first
  • Are opening the link on your iPhone (not a computer)
  • Are signed into the App Store with your Apple ID
