iPhone Installation

Follow these steps to install and configure the OwnTracks app on your iPhone for race tracking.

Password Required: You will need the server password from the race organisers before you can start tracking.

Installation

1

Install OwnTracks

Download the free OwnTracks app from the App Store:

Or search for "OwnTracks" in the App Store.

Configuration

2

Open OwnTracks Settings

Open the OwnTracks app and tap the (i) button in the top-left corner to access settings.

3

Set Connection Mode

Tap "Mode" and select "HTTP".

IMPORTANT: You must select HTTP not MQTT!
4

Configure Connection

Configure the settings as shown in this screenshot:

Setting Value
Mode HTTP (not MQTT!)
URL https://wstracker.org/api/owntracks
DeviceID Your sail number or name (e.g., S42 or John) - not the long UUID!
UserID Your name or identifier
Authentication ON (green)
Password Enter the password from the race organisers
DeviceID: Change the DeviceID from the long random UUID to something short and recognizable like your sail number. This is how you'll appear on the tracking map.
5

Configure Tracking Mode

Go back to settings and tap "Configuration". Set:

Setting Recommended Value
Monitoring Move (updates while moving)

The "Move" mode provides good position updates while conserving battery.

6

Grant Location Permission

When prompted, allow OwnTracks to access your location:

  • Select "Always Allow" for best tracking
  • This allows tracking to continue when the app is in the background
7

Start Tracking

Return to the main map screen. OwnTracks will automatically send your position to the server.

You can tap the publish button (arrow icon) to manually send your current position.

Testing Your Setup

Verify It's Working:
  1. In OwnTracks, tap the publish button to send your position
  2. Open https://wstracker.org in Safari or any web browser
  3. You should see your position appear on the map
  4. Your ID will be based on your User ID setting
Check for Errors:

If your position doesn't appear, check the OwnTracks status. Tap the (i) button and look at the connection status. Common issues:

  • 401 Unauthorized - Password is incorrect
  • Connection error - Check the URL is exactly as shown above

During the Race

  • Keep OwnTracks active - it will track in the background, but avoid force-closing it
  • Move mode is automatic - the app detects when you're moving and increases update frequency
  • Manual updates - tap the publish button anytime to send your current position immediately
  • Battery - OwnTracks is battery-efficient, but consider a power bank for long races

Troubleshooting

Position not appearing on map

Check these settings:

  • Mode is set to HTTP (not MQTT)
  • URL is exactly: https://wstracker.org/api/owntracks
  • Password is correct (ask race organisers)
  • Location permission is set to "Always"

Updates are infrequent

OwnTracks uses iOS location services which may limit updates when stationary. Try:

  • Set Monitoring mode to Move
  • Manually tap the publish button to force an update
  • Make sure the app has "Always" location permission

App shows "Unauthorized" error

The password is incorrect. Double-check with the race organisers and re-enter it in the Connection settings.
