Android Installation

Follow these steps to install and configure the Windsurfer Tracker app on your Android phone.

Event & Password Required: You will need to know which event you're tracking and the password for that event. Ask the race organisers for these details.

Installation

1

Download the APK

Download the Windsurfer Tracker app:

Download Tracker APK

2

Allow Installation from Unknown Sources

When you try to install, Android will ask you to allow installation from unknown sources. This is required because the app is not from the Play Store.

Tap "Settings" when prompted, then enable "Allow from this source".

3

Install the App

Open the downloaded APK file and tap "Install". Once installed, tap "Open" to launch the app.

Configuration

4

Open Settings

Tap the "Settings" button (gear icon) in the app.

5

Select Your Event

Tap "Select Event" to see available events. Choose the event you're participating in.

If you don't see your event, check with the race organisers that it has been created.

6

Enter Your Details

Configure the following settings:

Setting Value
Your Name Your name or sail number (e.g., John or S42)
Role Select Sailor (or Support/Spectator if applicable)
Password Enter the event password from the race organisers

Tap "Save" when done.

7

Grant Permissions

The app needs location permission to track your position. When prompted:

  • Tap "Allow" for location access
  • Select "Allow all the time" for background location (recommended)
  • Allow notifications (optional but helpful)
8

Start Tracking

Tap the large "Start Tracking" button. The app will begin sending your position to the server every 10 seconds.

Testing Your Setup

Verify It's Working:
  1. Start tracking on your phone
  2. Open https://wstracker.org in a web browser
  3. Select your event from the list
  4. You should see your position appear on the map within a few seconds
  5. The "ACK Rate" on your phone should show a percentage (ideally 80-100%)

During the Race

  • Keep the app running - don't close it or the tracking will stop
  • Lock screen is OK - the app continues tracking in the background
  • Request Assistance - if you need help on the water, long-press the green "Request Assistance" button
  • Battery - the app is designed to be efficient, but consider bringing a power bank for long races

Troubleshooting

ACK Rate shows 0%

This means the app cannot reach the server. Check:

  • You have selected the correct event
  • The event password is correct
  • You have mobile data or WiFi connection

Position not showing on map

Make sure:

  • Location permission is set to "Allow all the time"
  • GPS is enabled on your phone
  • You're outdoors with a clear view of the sky
